Building a Simple Chat App With Elixir and Phoenix

In the past few years, more and more applications have been transitioning to websockets for real-time communication, even forcing some frameworks to implement them (such as ActionCable in Rails 5). The Phoenix Framework for Elixir implements it natively, without depending on any external programs.

Today, we’re going to build a super simple chat application in Elixir using the Phoenix Framework. We’re going to ignore authentication, authorization and other features so we can quickly go over the basics, and get websockets in Elixir running.

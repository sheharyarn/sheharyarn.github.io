Installing TSOCKS (TorSocks) on MacOSX

I recently came across this nifty little terminal app called TSocks that allows you to use a Socks Proxy for your terminal commands. I already use TOR for all other web communications on Macbook Air, I wanted to use terminal over it as well. You can download and install it via MacPorts, Homebrew or manually. Sadly, the TSocks Project is no longer maintained. But after a little searching, I found TorSocks. Since TOR was the reason I wanted to use TSocks in the first place, TorSocks was perfect for this. I didn’t need to customize anything, I just installed and started using it.
